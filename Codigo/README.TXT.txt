Especificaciones:

-Para importar el proyecto, importar carpeta Gordinos. Si proyecto android tira error
se debe importar haciendo import->Android Project y se debe revisar en el build path que apunte
a el proyecto principal Gordinos

-Para poder compilar el codigo, revisar en Build Path la ruta a la libreria Kryonet, 
esta esta dentro de Gordinos/kryonet-2.20/jars/debug/onejar/kryonet-debug-2.18-all 
es OBLIGATORIO cambiar la ruta.

-Para jugar es OBLIGATORIO que haya un servidor y 2 clientes o si no no comenzara el juego

-El cambio de imagenes se deben hacer 2 clicks para que se vea la imagen
